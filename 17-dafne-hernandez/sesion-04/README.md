# Datos Dinámicos y *Variables*  

### Posición del mouse    
Estos determinan la posición del mouse en las coordenadas X e Y.  

```(mouseX,mouseY);```  

### Crear variables propias    

Esto es posible de hacer con el cómando let y const, para declarar una variable dinámica se debe usar ```let``` y para variables constantes ```const```  

### JavaScript Objects  

Sirve para organizar el código de una forma adecuada y ordenada, es la forma de agrupar muchas variables dentro de una variable.  

 ### random(); fuction    

El trabajo que tiene esta variable es volver un número aleatorio dentro de un rango que nosotros definimos.  
*ejemplo*    
- random() Si no pones nada, devuelve un número decimal entre 0 y 1.  
- random(máximo) Devuelve un num decimal entre 0 y el max que se elija.  
- random(mínimo, máximo) Devuelve un num decimal entre esos dos valores.  

### map(); fuction:  
Esta nos permite convertir un valor de un rango a otro, toma un número que esta en una escala y lo traduce proporcionalmente a una escala nueva.  

```map(valor, min_original, max_original, min_nuevo, max_nuevo);```
