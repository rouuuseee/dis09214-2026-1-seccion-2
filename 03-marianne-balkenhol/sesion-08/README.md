- finitud
- definicion

# Arrays

## Que es
Un array es una lista de datos guardados bajo un mismo nombre.

Cada elemento tiene una posicion numerada.

ideas:
- empiezan desde 0
- pueden guardar numeros, texto, variables y mas

## Para que sirve
- ordenar informacion
- guardar muchos datos
- repetir patrones
- trabajar con multiples objetos

## Idea 
En vez de crear muchas variables separadas, se guarda todo dentro de una lista.

---

# Class

## Que es
Una class es un molde para crear objetos.

Define:
- propiedades
- datos
- comportamientos


Una clase no es el objeto final, sino las instrucciones para crearlo.
copias infinitas indpendientes basadas en ese

Se usa para hacer muchas copias parecidas.

PALABRA CLAVE: NEW


Parte de la class que define los datos iniciales del objeto.

Ejemplo de cosas que puede guardar:
- posicion
- tamaño
- velocidad
- color

SINTAXIS:

1. palabra clave **class** + nombre q se le quiere dar
2. El método constructor
(donde se definen las
propiedades del objeto
usando .**this**)
3. Las funciones
personalizadas que definen lo
que hace el objeto.

---


Funciones dentro de una class.

Sirven para definir acciones del objeto.

Ejemplos:
- mover
- mostrar
