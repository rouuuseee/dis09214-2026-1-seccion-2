## BREVE HISTORIA DE LA COMPUTACIÓN   

   1. **Arquitectura del pensamiento**
- 1822: Máquina diferencial(Charles Babbage) Era como una calculadora gigante

    *   Nacimiento de la automatización

    *   “LA PRIMERA COMPUTADORA DEL MUNDO”
  
    *   Babbage diseñó esta máquina para sistematizar el cálculo y la tabulación de funciones polinómicas, tareas que hasta entonces se realizaban manualmente y eran propensas al error.

- 1991: Máquina diferencial II, la máquina de Charles pero más completa

    *  Funcionó perfectamente, demostrando que su diseño era correcto y que la máquina podría haberse construido en la época victoriana si la gestión hubiera sido diferente.

- 1834 - 1837: Máquina analitica (Charles Babbage)

    *  Dicen que la que fue la de la idea fue ada lovelace(Primera programadora)

    *  Fue diseñada para realizar cualquier tipo de cálculo establecido anteriormente y para tener una capacidad de análisis aún mayor que la Máquina Diferencial original. Se considera la primera máquina de cálculo completamente automática.

- 1801: Telar de jacquard, Primer sistema de almacenamiento de información binaria

    *  El telar usaba un sistema de tarjetas perforadas para lograr diseños o dibujos complejos y automatizados.

    *  Introduce 3 conceptos fundamentales: 

a) Separación entre software y hardware: 
* Hardware: El telar en sí (la máquina física, los engranajes, los hilos).
* Software: Las tarjetas perforadas.
* Antes de Jacquard, si queríamos que una máquina hiciera algo diferente, teníamos que reconstruirla o cambiar sus piezas. Con Jacquard, la máquina no cambia; solo cambias las tarjetas (el programa) y obtienes un resultado totalmente distinto.

b) El sistema binario: La ausencia o presencia de materia (agujero o cartón) determinaba una acción.

c) El pixel y la imagen rasterizada: La tarjeta perforada opera como el archivo digital (el .jpg), enviando las instrucciones precisas sobre qué puntos activar para componer la imagen visual.

- Ada lovelace: “Primera programadora”, condesa de lovelace, matemática y escritora británica, ella comprendió el potencial de las tarjetas  de la maquina analitica

    *  El primer algoritmo: Tradujo el trabajo de luigi menabrea y le agrego cosas, creando un algoritmo en papel para pasar a un computador.

    *  Las cartas de Lady Ada Lovelace demuestran que fue ella quien realmente entendió su potencial revolucionario. Mientras Babbage veía una forma de introducir números, Ada vislumbró que esas tarjetas permitían a la máquina manipular cualquier símbolo —incluso música o arte—, inventando así conceptualmente la programación que conocemos hoy.

- 1936: Máquina de turing: Es un experimento, máquina mental, Alan turing

    *  Define la Computabilidad Universal. Turing demostró que una máquina simple, con las instrucciones correctas, puede simular a cualquier otra máquina

   3. **Democratización del creative coding**
- Software libre(1983): Buscaban 4 libertades esenciales:
     *  1998 manifiesto: Uso, distribución, estudio y mejora 
  
      3.1 Uso: La libertad de ejecutar el programa como se desee, con cualquier propósito.
      3.2 Estudio: La libertad de estudiar cómo funciona el programa y cambiarlo para que haga lo que el usuario quiera. El acceso al código fuente es una condición necesaria.
      3.3 Distribución: La libertad de redistribuir copias para ayudar a otros.
      3.4 Mejora: La libertad de mejorar el programa y hacer públicas las mejoras, para que toda la comunidad se beneficie. El acceso al código fuente es necesario.

# Clase-20-03 MARKDOWN
- Github es una plataforma para almacenar, gestionar y compartir código(red social de programadores)
- Markdown es un lenguaje de marcado ligero, forma de escribir texto con formato(negrita, titulos, listdo, etc.)
- Para separar lineas o parrafos: Doble clic + enter
- Para hacer una linea: --- + enter
    * Ejemplo:
      ---
      
- Para titulos: 
  # titulo
  ## titulo 2
- Formato de texto:
    * Negrita: **texto** o __texto__
    * Cursiva: *texto* o _texto_
    * Negrita y cursiva: ***texto***
    * Tachado: ~~texto~~ (option + *)
    * Escapado de simbolos:  \#texto (option + ?)
- Viñetas: 4 espacios + * + 1 espacio + texto
    * Ejemplo
- Números: 4 espacios + numero mas punto + 1 espacio
    1. Elemento 1
    2. Elemento 2
       1. subcategoría
- Checkbox: 4 espacios + - + espacio + [ espacio ] + 1 espacio + texto
    - [ ] hola
    - [x] hola tachado
