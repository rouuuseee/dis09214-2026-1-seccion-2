# GitHub

## ¿Qué es GitHub?

GitHub es una plataforma en la nube diseñada para desarrolladores que permite almacenar, gestionar y compartir código utilizando Git, un sistema de control de versiones.

GitHub funciona como una red social para programadores, permitiendo colaborar con otras personas, compartir proyectos, guardar versiones de código y registrar cambios en proyectos públicos o privados.

GitHub se utiliza principalmente para:
- programación,
- documentación,
- portfolios,
- proyectos colaborativos,
- almacenamiento de código.

# Markdown

## ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que permite escribir texto con formato usando símbolos simples del teclado.

Markdown sirve para:
- crear títulos,
- escribir texto en negrita o cursiva,
- agregar imágenes,

Markdown es ampliamente utilizado dentro de GitHub para crear README y documentación.

---

# Cómo escribir en Markdown

## Saltos de línea

### Nuevo párrafo

Para crear un nuevo párrafo se debe presionar Enter dos veces.

Ejemplo:

```md
Primer párrafo.

Segundo párrafo.
```

---

### Bajar una línea sin crear párrafo

Se deben agregar dos espacios al final de la línea y luego presionar Enter.

Ejemplo:

```md
Texto línea 1··
Texto línea 2
```

---

### Forzar salto de línea

También se puede usar:

```html
<br>
```

# Sangrías

Las sangrías se hacen utilizando:
- TAB
- 4 espacios

Sirven para crear jerarquías y subcategorías.

---

# Encabezados

Los encabezados se crean utilizando `#`.

Mientras más `#`, menor es el nivel del título.

```md
# Título nivel 1
## Título nivel 2
### Título nivel 3
#### Título nivel 4
##### Título nivel 5
###### Título nivel 6
```

Importante:
Debe existir un espacio entre el símbolo y el texto.

# Formato de texto

## Negrita

```md
**Texto**
```

Resultado:

**Texto**

---

## Cursiva

```md
*Texto*
```

Resultado:

*Texto*

---

## Negrita y cursiva

```md
***Texto***
```

Resultado:

***Texto***

---

## Tachado

```md
~~Texto~~
```

Resultado:

~~Texto~~

---

## Escapar símbolos

Para escribir símbolos especiales sin activar Markdown se usa `\`.

Ejemplo:

```md
\#texto
```

Resultado:

\#texto

---
