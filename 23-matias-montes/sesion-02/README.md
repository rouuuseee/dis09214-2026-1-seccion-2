# Introducción

La presentación recorre la evolución de la computación desde las primeras máquinas mecánicas hasta el creative coding y las herramientas digitales contemporáneas, relacionando tecnología, arte y diseño.


## 1. Orígenes de la Computación

Charles Babbage

Máquina Diferencial (1822)

* Automatizaba cálculos matemáticos.
* Primera aproximación a una computadora mecánica.

Máquina Analítica (1834–1837)

Considerada el antecedente conceptual de las computadoras modernas.

## Introdujo:

* Procesador (The Mill)
* Memoria (The Store)
* Programación mediante tarjetas perforadas

  

## Telar de Jacquard (1801)

Primer sistema basado en tarjetas perforadas.

Aportes fundamentales

* Separación entre hardware y software
* Uso de lógica binaria (0 y 1)
* Antecedente de la imagen digital y los píxeles



## Ada Lovelace

Considerada la primera programadora de la historia.

Aportes

* Escribió el primer algoritmo computacional.
* Entendió que las máquinas podían manipular símbolos, música e imágenes, no solo números.



## Alan Turing

Máquina de Turing (1936)

Modelo teórico de computación universal.

Idea clave

Cualquier máquina puede ser simulada mediante instrucciones y código.

## 3. Computación y Arte

Arte Computacional (1960s)

Artistas comenzaron a usar algoritmos y computadoras para crear obras visuales.

Conceptos

* Arte generativo
* Algoritmos visuales
* Computación aplicada al arte



Sketchpad — Ivan Sutherland

Primer sistema de dibujo interactivo.

Introdujo

* Interfaces gráficas
* Manipulación directa
* Bases del diseño digital y CAD



Douglas Engelbart

Desarrolló conceptos fundamentales de las interfaces modernas:

* Mouse
* Ventanas
* Bitmapping



## 4. Software Libre y Creative Coding

Software Libre — Richard Stallman

Movimiento iniciado en 1983 con GNU.

Principios

* Libertad de uso
* Acceso al código
* Compartir y modificar software



Design By Numbers — John Maeda

Lenguaje creado para enseñar programación a diseñadores.

Idea principal

La programación como herramienta creativa y material artístico.



Processing (2001)

Creado por Casey Reas y Ben Fry.

Objetivo

Democratizar la programación visual para:

* artistas
* diseñadores
* principiantes



### p5.js

Biblioteca basada en Processing para programación creativa en la web.

Características

* Código abierto
* Accesible
* Orientada al arte interactivo



### Conclusión

La computación evolucionó desde máquinas mecánicas hasta sistemas creativos y visuales.

La programación dejó de ser solo una herramienta técnica y pasó a convertirse en:

* un medio artístico
* una forma de pensamiento
* una herramienta de creación cultural y visual
