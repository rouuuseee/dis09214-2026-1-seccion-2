# Pensamiento Computacional  

## Sus cuatro pilares  
### 1.- Descomposición  
*"Dividir para conquistar"* Se toma un problema grande y complejo y se rompe en partes más pequeñas y manejables.  
### 2.- Reconocimiento de patrones  
*"Encontrar similitudes"* Se observan tendencias o regularidades dentro de un problema, si algo se repite o sigue una lógica constante, se puede automatizar.  
### 3.- Abstracción  
*"Lo importante vs. el detalle"* Se filtra la información innecesaria y se queda solo con las características que definen el problema. Es crear una representación simbólica de la realidad.  
### 4.- Algoritmos  
*"La receta paso a paso"* Diseño de una serie de reglas ordenadas para resolver el problema. Es el "plan de acción" que debe seguir el sistema.  

## Tipos de interacción  
### 1.- Interacción discreta (eventos)  
Ocurre un evento específico (clic) y el sistema responde con una ación única (aparecn circulos). Es un interruptor de "encedido/apagado" o "acción/reacción".  
### 2.- Interacción Continua (Input de datos)  
Sistema reacciona constantemente al movimiento o estado del usuario, sin necesidad de hacer algo específico (clic).
